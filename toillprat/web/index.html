Paste from Character.AI or another app

Character.AI can't be exported, so copy the name, greeting and description off its page and paste them here. Pasted character JSON is understood too.

Or import a character card (.json or .png) from SillyTavern or a Character.AI card exporter.

Settings

This is the "brain" that writes the replies. All friends use it unless a friend has its own model. If replies stop working, pick another one.

"This device's voice" speaks replies instantly using the tablet's own voice — no wait. "Custom voices" uses your own voice server for each friend's chosen voice. "Online voices" uses OpenRouter's hosted voices — no voice server of your own, just the same key the replies already use.
